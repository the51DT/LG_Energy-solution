B.once

概述

LG能源解决方案的一次性电池评估服务，无需拆卸电池即可快速便捷地评估剩余性能和寿命。只需 5–30 分钟，即可为二手车交易或常规检查提供准确诊断，支持可靠的车辆价值评估和更快速的检测。

电池评估结果可以通过多种方式交付给客户

B-Once 应用程序 诊断结果可通过应用即时查看，并通过手机或平板电脑传递给用户。
API 通过 API 集成，将结果直接传输到您的服务、平台或数据库，供用户访问。
Mail 通过 API 集成，将结果直接传输到您的服务、平台或数据库，供用户访问。

评估方法

一次性评估服务只需将诊断设备连接到车辆并执行一个简单操作，即可即时生成证书。凭借速度与便捷性，该服务能够对电池状况提供高度准确的评估。

QuickScan （即时诊断）

基于车辆收集的 BMS 数据以及 LG能源解决方案的海量大数据，此电池评估方法可生成准确的健康状态 (SoH) 估算。

速度评估最快仅需 2 分钟完成

简便性无需拆卸电池

高效性一台设备即可评估多种车型

评估方法将诊断扫描仪插入车辆并打开/关闭点火开关

评估时间最少 2–5 分钟

准确性± 5%

使用案例

只需一次扫描，B-Once 就能快速、轻松且准确地评估电动车电池的当前状态——包括健康状态 (SoH)、电压、温度和故障信息——因此在二手车交易以及日常维护和检查中非常有用。
